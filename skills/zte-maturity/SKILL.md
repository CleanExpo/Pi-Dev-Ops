# ZTE Maturity

## Level 1: In the Loop
You type every prompt. You review every output. 2-5x advantage.

## Level 2: Out of the Loop
You write the spec. Agent executes AFK. You review the PR. 10-50x advantage.

## Level 3: Zero Touch Engineering
System detects work, writes specs, executes, tests, reviews, deploys. 100x+ advantage.
