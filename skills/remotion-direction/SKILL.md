# remotion-direction

Directs the video so it is not a generic slide deck.

## Direction rules

- One promise in the first 5 seconds.
- One audience and one job-to-be-done.
- Each scene must change the viewer's belief or move them toward the CTA.
- Motion supports comprehension; it does not decorate dead space.
- Avoid vague hype. Use concrete business outcomes.

## Verification checklist

- [ ] Hook is specific.
- [ ] Scene order is logical.
- [ ] CTA is visible and voiced.
- [ ] Visual direction matches the selected brand.
