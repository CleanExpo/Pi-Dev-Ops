# AI Generation Tool Shortlist

## Cost Rule

Use only tools Phill already has, tools already connected in this environment, or free/included tiers that do not create new spend. Unverified paid tools are research-only.

## Use First If Already Available

### Existing OpenAI / Codex Image Generation

Use for:

- source stills
- visual exploration
- keyframe drafts
- thumbnail concepts

Risk:

- Final claims, text, UI, and logos still need QA.

### HeyGen Connector

Use for:

- avatar/presenter/video-agent workflows
- short video generation where the connected account supports it

Risk:

- Not a fit for product UI truth unless used only as presenter/context layer.

### Existing Artlist / Stock / Audio Access

Use for:

- music
- SFX
- stock inserts

Risk:

- Every asset still needs a proof record.

## Access-Gated Research Only

### Adobe Firefly

Use for:

- commercially safer image/video generation
- image-to-video source tests
- AI editor-style assembly research
- audio/color/precision adjustment research

Why it matters:

- Best first choice where commercial safety and creative workflow control matter more than raw cinematic extremity.

Risk:

- May look cleaner/corporate rather than gritty and physically real.
- Do not use unless existing Adobe/Firefly access is verified.

### Runway Gen-4

Use for:

- controlled image-to-video
- premium motion from source frames
- reference-driven shot continuity
- Act-Two style performance/video workflows if a human-performance layer is introduced later

Why it matters:

- Strong candidate for turning carefully directed still frames into market-grade motion.

Risk:

- Can still drift on logos, text, UI, and precise product claims. Product UI must be composited/overlaid, not generated.
- Do not use unless existing Runway access is verified.

### Luma Ray

Use for:

- image-to-video
- video-to-video modify workflows
- preserving a source clip while improving style, camera, or environment

Why it matters:

- Good candidate when the team has a rough clip/reference and wants a more cinematic version without starting from text.

Risk:

- Video modification needs strict prompts specifying what must stay unchanged.
- Do not use unless existing Luma access is verified.

## Secondary Tools

### Kling 3.0

Use for:

- realism tests
- cinematic motion tests
- high-impact non-product inserts

Risk:

- Rights/commercial-use clarity and existing access must be verified before production use.

### CapCut

Use for:

- fast social editing
- captions
- 9:16 cutdowns
- background removal
- music integration

Risk:

- Good for packaging; not trusted as source of product truth.
- Do not use unless already installed/subscribed/free for the required export.

### Descript

Use for:

- transcript-led edits
- voice cleanup
- dubbing/localization if the campaign later needs variants

Risk:

- Less useful for cinematic image-to-video production.
- Do not use unless existing subscription is verified.

### Topaz Video AI

Use for:

- final upscale
- denoise
- sharpening
- stabilization
- reducing weak AI-video artefacts

Risk:

- Enhancement can create a plastic look if pushed too hard.
- Do not use unless already owned/subscribed.

Use for:

- source stills
- visual exploration
- image edits
- practical layouts

Access:

- Use only if already included in Phill's current Google/Gemini subscription.

### Ideogram

Use for:

### Firefly Image

Use for:

Access:

- Use only if existing Adobe/Firefly access is verified.

### Midjourney

Use for:

Do not use for:
