# Licensing Proof Sheet

## Rule

No asset enters the final cut unless its rights are documented.

## Required Records

For every external music, SFX, footage, or template asset:

- Provider.
- Asset title.
- Creator/artist.
- Download URL.
- Licence type.
- Download date.
- Project name.
- Allowed usage.
- Expiry/continuity note.
- Local file path.

## Current Asset State

- Existing voiceover: generated/local MP3s in `remotion-studio/public/audio/ra-wave-1-launch-poc/`.
- Existing UI/screencaps: committed under `remotion-studio/public/ra-wave-1-screencaps/`.
- Render output: ignored generated file under `remotion-studio/output/`.

## Blocking Rule

Any stock shot with recognisable people, real business signage, customer identity, or unclear commercial licence blocks publish until replaced or cleared.
