# Judge Evidence Ranking

Judge uses this source hierarchy:

1. Official vendor docs
2. Official SDK/API references
3. Official changelogs
4. Repo source code
5. Tests, traces, logs, schemas, migrations, CI
6. Standards/specs
7. Known expert material
8. Blogs/videos/social posts only as discovery leads
9. LLM memory is not evidence

Any claim without support must be marked `UNSUPPORTED`.
